# | Programação em Python para Data Science

Repositório com exercícios e práticas desenvolvidas durante meus estudos de Data Science no SENAI, utilizando Python e a biblioteca Pandas.



## | Sobre

Este repositório reúne atividades práticas voltadas para manipulação, tratamento e análise de dados utilizando Python.

Os exercícios abordam desde conceitos básicos até aplicações em análise de dados, com foco no uso da biblioteca Pandas.



## | Objetivo

- Aprender análise de dados com Python  
- Utilizar a biblioteca Pandas  
- Praticar manipulação e limpeza de dados  
- Desenvolver raciocínio analítico  



## | Tecnologias utilizadas

- Python  
- Pandas  
- Google Colab  



## | Conteúdos estudados

Durante os exercícios, foram abordados:

- Carregamento de dados  
- Limpeza e tratamento de dados  
- Remoção de duplicados  
- Manipulação de colunas  
- Estruturas condicionais  
- Operadores aritméticos  
- Manipulação de strings  
- Conjuntos (sets)  
- Tuplas  
- Criação de gráficos  
- Introdução à análise preditiva  



## | Estrutura do Projeto

- `Carregando_Dados.ipynb` → Leitura de dados  
- `DataPrep_Duplicados.ipynb` → Tratamento e limpeza  
- `Criando_Gráficos.ipynb` → Visualização de dados  
- `Análise_Preditiva.ipynb` → Introdução à análise de dados  

- `Estrutura_condicional_IF.ipynb` → Condições  
- `Operadores_Aritméticos_03.ipynb` → Operações matemáticas  
- `f_string.ipynb` → Formatação de strings  
- `Sets.ipynb` → Conjuntos  
- `Tuplas.ipynb` → Tuplas  

- `1Mão_na_massa.ipynb` e `2Mão_na_Massa.ipynb` → Exercícios práticos  



## | Destaque

Este repositório demonstra a aplicação prática de conceitos essenciais de Data Science, com foco em preparação e análise de dados utilizando Pandas.



## | Evolução

Este projeto representa minha evolução no aprendizado de análise de dados, desde conceitos básicos até aplicações mais estruturadas.
